Featured Project — Eventore

Open-source multi-protocol streaming console — connect, browse, live-stream, and administer every broker from one UI.

Creator · Apache 2.0 · v0.1.0

One console for every stream

Eventore is a modular control plane with a React console for Kafka, Pulsar, RabbitMQ, MQTT, JMS, AWS Kinesis, GCP Pub/Sub, and Azure Service Bus — with cloud presets for MSK, Event Hubs, and more. Born as an Amdocs innovation tool; now open source on GitHub with Helm charts and GHCR images.

  • Spring Boot 3.3 backend — OpenAPI 3, WebSocket/SSE live streaming, per-protocol Maven modules
  • Kafka depth: consumer groups, lag, message search, topic admin, ACLs, produce with headers
  • ADMIN / DEV / READONLY deployment modes · Kubernetes-ready Helm on OCI
  • Optional MCP server for AI agents — probe brokers, publish, and inspect without the UI

Featured Writing — Agent Engineering Framework

Beyond prompt engineering: a practitioner framework for production-grade AI agents.

Author · CC BY 4.0 · v1.2 · 35 citations

The 10 Pillars of Production Agent Engineering

Agent Engineering reframes production agents as a systems problem — a deterministic envelope around a probabilistic core. Ten coordinated disciplines (Prompt, Goal, Trajectory, Context, Memory, Tool, Intelligence, Execution, Governance, Evaluation) with case studies, a readiness rubric, and a design checklist.

  • Three worked case studies — telecom RCA, billing disputes, automated software maintenance
  • Six-phase lifecycle: specify, design, build, evaluate, deploy, operate
  • Dual-evaluation principle — grade outcomes and trajectory quality
  • Production readiness rubric — score any agent 0–5 per pillar before deploy
